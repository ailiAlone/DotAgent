- Bug 修复记录：1) player.gd 的 heal() 不应 emit hit（之前错误地连接到了 _on_player_hit，导致 lives 减 1，HUD 显示血包后下降）；2) _on_player_hit 不应减 lives（只在 died 时减）；3) _game_over 中不应设置 process_mode=DISABLED，否则 add_child 的 GameOver 节点继承此状态，tween 不跑，界面永远不可见。要让 GameOver 始终可见，需 go.process_mode = PROCESS_MODE_ALWAYS。HUD 现在显示 ♥♥♥ ♡ LIVES 3（船血量+生命数）。
- GameOver 居中显示修复：Godot 中 Control 节点被加到 Node2D 父节点下时，anchor 计算失效（因为 Node2D 没有 viewport size），Control 退回到 (0,0) 位置 + 默认 64×64 大小。修复方案：把 GameOver 加到 root（不是 game Node2D），并强制 size = viewport size（fallback 到 project setting 1280×720）。同时 top_level = true 让 GameOver 脱离父节点变换影响。GameOver retry/menu 回调通过 _find_main() 向上查找 Main 节点添加新场景。
